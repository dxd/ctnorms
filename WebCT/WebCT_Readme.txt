Welcome to Colored Trails Web Application:

make sure you download the latest rev from the svn repository:
https://viki.eecs.harvard.edu/repos/ct3/branches/BGU/WebCT

Deployment instructions:

1. make sure you have a configured and running apache tomcat server.
2. copy the war/WebCT.war file to the apache webapps folder.
3. wait for apache tomcat to deploy the web app (you will notice a new folder called WebCT created.
4. navigate to webapps\WebCT folder and copy the webct.ini file to c:\.
5. webct.ini file can be edited to change game settings without re-compiling.
